Part A: Visualization of Activation Functions

In this part, the Sigmoid, Tanh, and ReLU activation functions were implemented and visualized using Python and Matplotlib. Their output ranges, saturation regions, gradient behavior, computational efficiency, and typical applications were analyzed. The comparison showed that ReLU is computationally efficient and widely used in deep neural networks, while Sigmoid and Tanh are useful for specific tasks but may suffer from the vanishing gradient problem due to saturation.

Part B: Performance Comparison of Activation Functions

An identical Artificial Neural Network (ANN) architecture was trained separately using Sigmoid, Tanh, and ReLU activation functions. The models were evaluated based on training accuracy, validation accuracy, training loss, validation loss, and the number of epochs required for convergence. The experimental results indicated that ReLU achieved faster convergence, higher accuracy, and lower loss compared to Sigmoid and Tanh, making it the most effective activation function for this experiment.

Part C: Comparison of Optimization Algorithms

The same ANN model was trained using four optimization algorithms: Stochastic Gradient Descent (SGD), Momentum, RMSProp, and Adam. Their performance was compared using training loss, validation loss, convergence speed, final training accuracy, and final validation accuracy. Loss and accuracy curves were plotted for each optimizer. Among all the optimizers, Adam demonstrated the fastest convergence, stable learning, and the best overall accuracy, while SGD required more epochs to achieve similar performance.

The deep learning experiments were carried out using Google Colab, and Google Drive was mounted to save notebooks and trained models securely. A detailed README.md file was prepared to document the experiment objectives, methodology, implementation, results, and conclusions, ensuring proper experiment management, reproducibility, and collaboration.
